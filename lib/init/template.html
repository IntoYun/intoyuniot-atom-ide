新建工程

本向导可以帮您给指定的开发板构建新的构建环境([env:...])。 也就是您可以实现跨平台构建，一份代码，多个嵌入式平台/开发板。
所有改变都将写入工程根目录下的配置文件intoyuniot.ini。
如果您不使用向导，也可以直接修改配置文件intoyuniot.ini。

首次初始化工程需要联网， 它将从网络安装toolchains, frameworks, SDKs。 请耐心等待工程初始化完成。

PlatformIO Project Configuration File "platformio.ini" allows to set:

  • Build options: build flags, source filter, extra scripting
  • Upload options: custom upload port, speed and extra flags
  • Library options: dependencies, extra library storages
Please visit documentation for the other options and examples http://docs.platformio.org/page/projectconf.html

The only first selected board (build environment [env:...]) will be used to generate data for Intelligent Code Completion and Smart Code Linter.
